# Network Simulation

Interactive web-based simulations for learning networking concepts. Built for personal study and visualization.

---

## Simulations


| File | Description |
|------|-------------|
| `network-simulation.html` | Basic network topology simulation |
| `nmap-wireshark-simulation.html` | Nmap scanning and Wireshark packet capture simulation |
| `osi-packet-simulation.html` | OSI model packet flow visualization |

---

## How to Use

Just open the `.html` files in your browser. No installation needed.

---

## Topics Covered

- Network topology basics
- Nmap scanning techniques
- Wireshark packet analysis
- OSI layer concepts and packet encapsulation
